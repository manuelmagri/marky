# marky

A Markdown editor for Windows.

## Overview and features
- **Local-First**. Your Markdown files are stored on your computer storage.
- **Privacy-First**. marky doesn't collect any data from you and your computer.

## Platforms and Installation
marky is available only for Windows (for now!). 

More informations about installation guide and new platforms are coming soon.
